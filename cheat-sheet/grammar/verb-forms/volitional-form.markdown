## Volitional Form (Casual)
let's do~
- Godan Verbs /u/ → /o/ + う
- Ichidan Verbs - る → よう
- Irregular Verbs
  - する → しよう
  - 来る → 来よう

| Godan Verbs   | Ichidan Verbs     | Irregular Verbs |
|:------------- |:----------------- |:--------------- |
| 話す → 話そう | 食べる → 食べよう | する → しよう   |
| 聞く → 聞こう |                   | 来る → 来よう   |
| 泳ぐ → 泳ごう |                   |                 |
| 飲む → 飲もう |                   |                 |
| 遊ぶ → 遊ぼう |                   |                 |
| 死ぬ → 死のう |                   |                 |
| 買う → 買おう |                   |                 |
| 待つ → 待とう |                   |                 |
| 直る → 直ろう |                   |                 |

## Volitional Form (Polite)
let's do~
- Verb (Stem Form) + ましょう

| Godan Verbs         | Ichidan Verbs         | Irregular Verbs   |
|:------------------- |:--------------------- |:----------------- |
| 話す → 話しましょう | 食べる → 食べましょう | する → しましょう |
| 聞く → 聞きましょう |                       | 来る → 来ましょう |
| 泳ぐ → 泳ぎましょう |                       |                   |
| 飲む → 飲みましょう |                       |                   |
| 遊ぶ → 遊びましょう |                       |                   |
| 死ぬ → 死にましょう |                       |                   |
| 買う → 買いましょう |                       |                   |
| 待つ → 待ちましょう |                       |                   |
| 直る → 直りましょう |                       |                   |

## Volitional Form + とおもう
plan, thinking of doing~

attempted to do~ (but unsuccessful)
