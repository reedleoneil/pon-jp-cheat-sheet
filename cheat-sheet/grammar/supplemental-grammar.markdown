## ばかり

**ばかり** only; nothing but~  
Noun + ばかり  
Verb (て Form) + ばかり

**たばかり** just finished; something just occurred  
Verb (た Form) + ばかり

---

## Embedded Question

**か** embedded questions  
 sentence + か + sentence

**かどうか** whether or not~  
Verb + かどうか  
Noun + かどうか  
い・Verbal Adjective + かどうか  
な・Noun Adjective + かどうか

---

## Expectations

**はずだ** it must be; it should be (expectation)  
Verb (Dictionary Form) + はず(だ)  
Noun + の + はず(だ)  
い・Verbal Adjective + はず(だ)  
な・Noun Adjective + な + はず(だ)  

**はずがない** cannot be (impossible)  
Verb (Dictionary Form) + はずがない  
Noun + の + はずがない  
い・Verbal Adjective + はずがない  
な・Noun Adjective + な + はずがない

い・Verbal Adjective - い + がる  
な・Noun Adjective + がる

**がり** someone tends to; has a tendency to; has a sensitivity to~  
い・Verbal Adjective - い + がり  
な・Noun Adjective + がり

## 方がいい

**ほうがいい** had better; it'd be better to; should~

## In Order To

**ように** in order to, so that~  
Verb (Dictionary Form・ない Form) + ように

**ために** for; in order to; for the benefit of; because of; as a result of~  
Verb + ため(に)  
Noun + の + ため(に)  
い・Verbal Adjective + ため(に)  
な・Noun Adjective + な + ため(に)
