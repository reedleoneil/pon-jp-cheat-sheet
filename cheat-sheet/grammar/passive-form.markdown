## Passive Form
passive form; passive voice

**Godan Verbs**  
/u/ → /a/ + れる

**Ichidan Verbs**  
る → られる

**Irregular Verbs**  
する → される  
来る → こられる

| Godan Verbs     | Ichidan Verbs       | Irregular Verbs |
|:--------------- |:------------------- |:--------------- |
| 話す → 話される | 食べる → 食べられる | する → される   |
| 聞く → 聞かれる |                     | 来る → 来られる |
| 泳ぐ → 泳がれる |                     |                 |
| 飲む → 読まれる |                     |                 |
| 遊ぶ → 遊ばれる |                     |                 |
| 死ぬ → 死なれる |                     |                 |
| 買う → 買われる |                     |                 |
| 待つ → 待たれる |                     |                 |
| 直る → 直られる |                     |                 |
