## Verb た Form
verb past form

- **Godan Verbs**  
  す → した  
  く → いた  
  ぐ → いだ  
  む → んだ  
  ぶ → んだ  
  ぬ → んだ  
  る → った  
  う → った  
  つ → った

- **Ichidan Verbs**  
  る → た

- **Irregular Verbs**  
  する → した  
  来る → きた  
  行く → 行った

| Godan Verbs   | Ichidan Verbs   | Irregular Verbs |
|:------------- |:--------------- |:--------------- |
| 話す → 話した | 食べる → 食べた | する → させた   |
| 聞く → きいた |                 | 来る → 来た     |
| 泳ぐ → 泳いだ |                 | 行く → 行った   |
| 飲む → 飲んだ |                 |                 |
| 遊ぶ → 遊んだ |                 |                 |
| 死ぬ → 死んだ |                 |                 |
| 買う → 買った |                 |                 |
| 待つ → 待った |                 |                 |
| 直る → 直った |                 |                 |

## Adjective た Form
adjective past form

verbial adjectives conjucate like verbs and noun adjectives conjucate like nouns  

- **い・Verbal Adjectives**  
  い → かった   
  ない　→ なかった

- **な・Noun Adjectives**  
  \+ だった
